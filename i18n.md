# 国际化 (i18n) 开发规范

## 概述

本项目使用 `@ngx-translate/core` 和 `ng-zorro-antd/i18n` 实现国际化支持，目前支持 **中文** 和 **英文** 两种语言。

## 目录结构

```
src/
├── assets/
│   └── i18n/
│       ├── zh.json      # 中文翻译
│       └── en.json      # 英文翻译
└── app/
    ├── layout/
    │   └── lang-selector/    # 语言选择器组件
    └── app.config.ts         # 国际化配置
```

## 核心规则

### 1. 翻译键命名规范

- 使用 **大写字母** 和 **下划线** 分隔单词
- 使用 **点号** 表示层级关系
- 保持中英文键结构一致

**推荐：**
```json
{
  "USER_MANAGEMENT": {
    "TITLE": "用户管理",
    "ADD_USER": "添加用户"
  }
}
```

**不推荐：**
```json
{
  "userManagement": "用户管理",
  "addUser": "添加用户"
}
```

### 2. 禁止硬编码

所有用户可见的文本必须使用翻译管道，不得硬编码在模板或 TypeScript 代码中。

```typescript
// ✅ 正确
imports: [TranslateModule]
```

```html
<!-- ✅ 正确 -->
<h1>{{ 'APP.TITLE' | translate }}</h1>
<button>{{ 'BUTTONS.SAVE' | translate }}</button>

<!-- ❌ 错误 -->
<h1>应用标题</h1>
<button>保存</button>
```

### 3. 保持翻译一致性

- **相同意义的文本使用相同的翻译键**：例如，"保存" 按钮在整个应用中都使用 `BUTTONS.SAVE`
- **避免硬编码文本**：所有用户可见的文本都应该使用翻译
- **及时更新翻译**：添加新功能时同步更新翻译文件

## 快速开始

### 在组件中使用

```typescript
import { Component, inject } from '@angular/core';
import { TranslateModule } from '@ngx-translate/core';

@Component({
  selector: 'app-my-component',
  imports: [TranslateModule],  // 必须导入
  template: `...`
})
export class MyComponent {
  private translate = inject(TranslateService);
}
```

```html
<!-- 简单使用 -->
<h1>{{ 'APP.TITLE' | translate }}</h1>

<!-- 带参数 -->
<p>{{ 'WELCOME_MESSAGE' | translate:{name: user.name} }}</p>

<!-- 在属性中使用 -->
<button [title]="'BUTTONS.SAVE' | translate">
  {{ 'BUTTONS.SAVE' | translate }}
</button>
```

### 在 TypeScript 中使用

```typescript
// 获取当前语言
const currentLang = this.translate.currentLang;

// 获取翻译文本
this.translate.get('MENU.DASHBOARD').subscribe((text: string) => {
  console.log(text);
});

// 即时获取（不推荐用于动态内容）
const text = this.translate.instant('BUTTONS.SAVE');
```

### 切换语言

项目已提供 `LangSelector` 组件，可直接使用：

```html
<app-lang-selector></app-lang-selector>
```

手动切换语言：

```typescript
import { inject } from '@angular/core';
import { TranslateService } from '@ngx-translate/core';
import { NzI18nService } from 'ng-zorro-antd/i18n';
import { StorageService, StorageType } from '../../core/services/storage.service';

export class MyComponent {
  private translate = inject(TranslateService);
  private i18n = inject(NzI18nService);
  private storageService = inject(StorageService);

  changeLanguage(lang: string): void {
    // 切换 ngx-translate 语言
    this.translate.use(lang);

    // 切换 ng-zorro-antd 语言
    if (lang === 'zh') {
      this.i18n.setLocale(zh_CN);
    } else if (lang === 'en') {
      this.i18n.setLocale(en_US);
    }

    // 保存用户偏好（使用 StorageService）
    this.storageService.setItem('preferredLanguage', lang, {
      type: StorageType.LOCAL,
      ttl: 30 * 24 * 60 * 60 * 1000 // 30天
    });

    // 刷新页面使变更生效（某些组件需要）
    window.location.reload();
  }
}
```

## 添加翻译的步骤

### 1. 添加翻译键值对

**在 zh.json 中添加：**

```json
{
  "NEW_MODULE": {
    "TITLE": "新模块",
    "DESCRIPTION": "新模块的描述",
    "ACTION": {
      "ADD": "添加",
      "EDIT": "编辑",
      "DELETE": "删除"
    }
  }
}
```

**在 en.json 中添加对应的翻译：**

```json
{
  "NEW_MODULE": {
    "TITLE": "New Module",
    "DESCRIPTION": "Description of new module",
    "ACTION": {
      "ADD": "Add",
      "EDIT": "Edit",
      "DELETE": "Delete"
    }
  }
}
```

### 2. 在组件中使用

```html
<h1>{{ 'NEW_MODULE.TITLE' | translate }}</h1>
<p>{{ 'NEW_MODULE.DESCRIPTION' | translate }}</p>
<button>{{ 'NEW_MODULE.ACTION.ADD' | translate }}</button>
```

## 最佳实践

### 1. 动态内容

对于参数化的翻译：

```json
{
  "WELCOME_MESSAGE": "欢迎，{{name}}！",
  "ITEMS_COUNT": "共 {{count}} 个项目"
}
```

```html
<p>{{ 'WELCOME_MESSAGE' | translate:{name: userName} }}</p>
<p>{{ 'ITEMS_COUNT' | translate:{count: itemCount} }}</p>
```

### 2. 复数处理

```json
{
  "MESSAGE_COUNT": "{{count}} 条消息",
  "MESSAGE_COUNT_PLURAL": "{{count}} 条消息"
}
```

```typescript
getMessageText(count: number): string {
  return this.translate.instant(count === 1 ? 'MESSAGE_COUNT' : 'MESSAGE_COUNT_PLURAL', { count });
}
```

### 3. 格式化处理

日期、数字、货币等格式化使用 Angular 的内置管道：

```html
<p>{{ user.birthday | date:'yyyy-MM-dd' }}</p>
<p>{{ price | currency:'CNY' }}</p>
<p>{{ quantity | number:'1.2-2' }}</p>
```

### 4. 文本长度考虑

不同语言的文本长度可能差异较大：
- 设计 UI 时预留充足空间
- 使用 CSS 的 `text-overflow: ellipsis` 处理超长文本
- 避免固定宽度的按钮或标签

## 常见问题

### 1. 翻译不生效

**检查项：**
- [ ] 是否正确导入了 `TranslateModule`
- [ ] 翻译键是否正确（大小写敏感）
- [ ] 翻译文件是否已添加对应的键值对
- [ ] 是否在正确的语言环境下测试

### 2. ng-zorro-antd 组件未翻译

**原因：** ng-zorro-antd 的国际化需要单独设置

**解决：** 确保调用了 `this.i18n.setLocale()` 并刷新页面

### 3. 添加新语言支持

如需添加新语言（如日语）：
1. 创建 `ja.json` 翻译文件
2. 安装对应的语言包
3. 更新 `app.config.ts` 注册新语言
4. 更新 `LangSelector` 组件

参考现有 `zh.json` 和 `en.json` 的实现。

## 开发检查清单

添加新组件或功能时：

- [ ] 规划翻译键（符合命名规范）
- [ ] 在 `zh.json` 中添加中文翻译
- [ ] 在 `en.json` 中添加英文翻译
- [ ] 导入 `TranslateModule`
- [ ] 模板中使用 `| translate` 管道
- [ ] 切换语言测试显示是否正确
